## 📝 快速填写指南

### 如何从输出中提取块号？

**输出格式:**
```
{12345} normal block at 0x00A1B2C3, 256 bytes long.
        D:\Project\file.cpp(123)
```

**提取方法:**
- 块号 = **12345** (花括号内的数字)
- 大小 = **256** (at前面)
- 文件 = **file.cpp** (路径的最后部分)
- 行号 = **123** (括号内的数字)

### 如何判断优先级？

```
大于1000字节  → ☐高 优先处理
100-1000字节  → ☐中 次要处理
小于100字节   → ☐低 可暂时忽略
```

## 💡 技巧

### 技巧1: 按大小排序泄漏块
```
1. 最大的块 (优先处理)
2. 中等的块 (次要处理)
3. 最小的块 (最后处理)
```

### 技巧2: 相同文件的泄漏块聚在一起
```
如果多个块来自同一文件
可能是同一个问题的表现
一次修复可能解决多个泄漏
```

### 技巧3: 记录修复历史
```
块12345 → CDialogBase.cpp → [已修复]
块12346 → CDialogBase.cpp → [已修复]
块12347 → libsparkusb.cpp → [需要第三方库修复]
```
